# IM
简单的仿QQ聊天安卓APP

该项目用JAVA的socket实现，包含客户端，服务器与数据库，如果想运行需要注意一下几点：

 1. 服务器更改配置文件dbcpconfig.properties
 2. 我已经将所需的依赖库也上传了，如果还缺少某些依赖库，需要你自己去下
 3. 注意客户端的端口号应与服务器的端口号一致。
 4. 客户端出现服务器异常时，请注意客户端的SERVER_IP，需要保证客户端ping通服务器
